The handoff tells an AI client or operator how to consume the bundle without guessing.

# Agent Handoff

Use `manifest.json` as the entry point.
Prefer markdown under `ready/`.
Use metadata files for provenance and file type notes.
When citing, reference `doc_id` and section heading.
Avoid using source PDFs directly unless verification is needed.
